# 🚀 Crypto Portfolio Management System

## 📌 Overview
This project helps users analyze cryptocurrency risk, generate investment portfolios, and predict future prices using machine learning.

## ⚙️ Features
- Risk Analysis (Low / Medium / High)
- Portfolio Allocation
- Crypto Price Trends
- AI Price Prediction
- Interactive Dashboard (Streamlit)
- Database Storage

## 🛠️ Tech Stack
- Python
- Pandas
- Streamlit
- Scikit-learn
- SQLite

## ▶️ How to Run

1. Run risk pipeline:
   python run_risk_pipeline.py

2. Run dashboard:
   streamlit run dashboard.py

## 📊 Output
- Risk Report
- Portfolio Allocation
- Price Prediction
